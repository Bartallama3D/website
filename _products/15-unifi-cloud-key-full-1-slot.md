Introducing our 1U Full-Width Steel Rackmount Bracket for UniFi Cloud Key G2 (1-Slot). This bracket allows you to mount your Cloud Key in a standard server rack. The brackets spans the full width of a 1U slot and supports mounting the cloud key on either the right or left side. The bracket is designed to be tight around your cloud key but also has both the top and bottom open to allow for open air cooling of the device.  This bracket features 1-slot for a Unifi Cloud Key and can only accomodate 1 unit, if you are looking to mount 2 units check out our 2-slot model!

\* Cloud Key is not included. Mounting hardware for standard server racks is included.
