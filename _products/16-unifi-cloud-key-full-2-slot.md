Introducing our 1U Full-Width Steel Rackmount Bracket for UniFi Cloud Key G2 (2-Slot). This bracket allows you to mount up to 2 Cloud Key Gen2 units in a standard server rack. The bracket comes with a cover plate to cover the second slot if you plan to add a second cloud key in the future. If you only need to mount a single Cloud Key Gen2 unit, check out our 1-slot version! The bracket is designed to be tight around your cloud key but also has both the top and bottom open to allow for open air cooling of the device.

\* Cloud Key is not included. Mounting hardware for standard server racks is included.
